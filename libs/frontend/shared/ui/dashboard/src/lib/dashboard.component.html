Hi and welcome!


My name is Ivan Rogulj. I am a software developer and a hobbyist musician.

This is a space where you can try out my online synths, VST plugins, PureData projects, Arduino projects and synth patches, as well as learn about various topics on music and programming in my blog articles.

Online synths can be connected to your MIDI device and played directly in your browser (Google Chrome has the best support for Web Audio API and Web MIDI API - which are used for making these synths, so it is recommended browser).


Audio DAW plugins are made in C++ JUCE framework, all compatible with popular DAWs like Ableton, FL Studio and LogicPro - and free for download (.vst3 and .au extensions for every plugin).


You will also find downloadable synth patches for Yamaha DX7 and Behringer DeepMind (.sysex extension).
There are PureData projects, as well (.pd extension).


In the blog section, you will find topics on Web Audio API, JUCE framework (C++ based) and PureData.
I will also write about non-music stuff, like programming in popular frontend/backend JS frameworks and libraries (Angular/NestJS/RxJS/Nx).

All of my projects are free for you to use and download.

Enjoy!
